～githubのターミナルで使用できるコマンド～

※これはすべてのファイルをステージする際に使用する（すべてではない場合はファイルを指定する必要がある）

git commit -m "ここにメッセージを書く"

git push origin ブランチ名
ブランチの確認方法:
git branch

pull(同期):
git pull origin ブランチ名
